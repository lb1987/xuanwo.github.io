POJ 2328 Guessing Game

题目

源地址：http://poj.org/problem?id=2328

理解

猜数字游戏，还是比较熟悉的模型。
先确定一个正确数字，另一个人猜数字，然后回答猜测数字比正确数字是太高、太低还是正确。现在题中给定猜测的过程，让我们用程序验证这个猜测过程中，回答是否正确。
